# How to invite

## Inviting goPBot

Adding goPBot to your server is a simple and easy process:

1\) Go to [invite.gopbot.xyz](https://invite.gopbot.xyz)

2\) Select your server and press continue&#x20;

3\) Press "Authorise"

{% hint style="success" %}
4\) You're done! Enjoy goPBot!
{% endhint %}
